# Advanced Customer Segmentation using Machine Learning

## 📌 Overview
This project applies machine learning techniques to segment customers based on their behavior and purchasing patterns. It helps businesses better understand their customers and create targeted marketing strategies.

## 🎯 Objectives
- Identify customer segments using clustering algorithms
- Analyze customer behavior and spending patterns
- Generate actionable business insights

## 📊 Dataset
Marketing Campaign Dataset including:
- Income
- Birth Year
- Product Spending
- Purchase Frequency

## ⚙️ Project Pipeline
- Data Exploration (EDA)
- Data Cleaning
- Feature Engineering
- Feature Scaling
- Clustering (KMeans, DBSCAN)
- Model Evaluation (Silhouette Score)
- Visualization (PCA)

## 🧠 Feature Engineering
- Customer Age
- Total Spending
- Total Purchases

## 🤖 Models Used
- K-Means Clustering
- DBSCAN Clustering

## 📈 Evaluation
- Silhouette Score

## 💡 Business Insights
- VIP Customers (High income & high spending)
- Discount Seekers (Low income & frequent purchases)
- Growth Opportunities (Moderate engagement)
- Upsell Opportunities (High income & low purchases)

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
